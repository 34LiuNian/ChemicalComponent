🔤 FontFile API Demo — 原生 C++ 字体自定义

原理： RDKit 的 MolDraw2D 支持 C++ fontFile() API，直接让 FreeType 用指定 .ttf 文件渲染原子标签。本 demo 使用 FS 构建（含 -sFORCE_FILESYSTEM=1）的 RDKit_minimal.js，通过 Emscripten 的 FS.writeFile() 将字体文件注入 WASM 虚拟文件系统， 然后在绘制选项中设置 fontFile: "/fonts/myfont.ttf"， 即可让 RDKit 从路径渲染原生字体——不再需要 JS 侧的 SVG 覆盖层 hack。
